Cómo funciona la herramienta

La herramienta de auditoría de sitios web del CEPD introduce una capa fácil de usar en el navegador Chromium para ayudar a evaluar de un vistazo si un sitio web recopila o almacena información en un navegador de manera conforme (esdecir,informando y recabando el consentimiento del usuario cuando sea necesario).

Todos los datos probatorios pueden almacenarse en las herramientas a través de un análisis que reúne diferentes escenarios, que representan una interacción (porejemplo, consentimiento, denegación, etc.) con el sitio web dado.

La evaluación se facilita con herramientas automatizadas, mientras que la decisión sobre el cumplimiento/incumplimiento la toma el funcionario encargado del caso.

La base de conocimientos puede ayudar a identificar rastreadores conocidos. La plantilla se puede utilizar para personalizar los informes para un escenario determinado.

A través de la barra de navegación lateral izquierda, puede acceder a todas estas funcionalidades:

Navegar Obtenga una visión general rápida de los datos personales almacenados o transferidos en o por el navegador. Los resultados no se almacenarán en la herramienta.
Nuevo Iniciar un nuevo análisis del sitio web para recopilar evidencia. Las pruebas se almacenan en la herramienta para una evaluación posterior de su cumplimiento de los requisitos legales.
Lista Lista de análisis almacenados en la herramienta con su evaluación.
Análisis > Escenario Sesión actual adjunta a un análisis y un escenario en este análisis. Se puede adjuntar evidencia al análisis y escenario referenciado hasta que se cierre esta sesión. Una vez cerrado, ya no se pueden adjuntar pruebas a este análisis y escenario.
Base de conocimientos La base de conocimientos enumera los rastreadores conocidos y sus propósitos correspondientes. Se puede acceder a esta información de forma dinámica mientras navega o evalúa el cumplimiento de un sitio web. Únicamente sirve como herramienta para facilitar el análisis por parte del funcionario encargado del caso.
Plantilla Las plantillas se utilizan para generar informes adaptados a necesidades específicas.
Configuración Personalice la información mostrada mientras navega o evalúa el cumplimiento de un escenario determinado.
Ayuda Referencia a la documentación de esta herramienta.

Como primera introducción, le recomendamos que eche un vistazo a la sección Proporcionar un nuevo análisis.
