Δημιουργία αναφορών

Οι εκθέσεις είναι μορφοποιημένα έγγραφα σύμφωνα με ένα δεδομένο υπόδειγμα πληροφοριών που περιέχονται σε μια ανάλυση, ένα σενάριο ή μια δεδομένη κάρτα.

Οι αναφορές επεξεργασίας διεπαφής είναι προσβάσιμες σε κάθε γραμμή του πίνακα Σύνοψη των αξιολογήσεων σε μια δεδομένη ανάλυση ή σενάριο.

Διεπαφή αναφοράς

Η διεπαφή αναφοράς αποτελείται από τις ακόλουθες τρεις κύριες συνιστώσες:

  • η γραμμή εργαλείων επεξεργασίας/εξαγωγής στο επάνω μέρος της διεπαφής επιτρέπει την επεξεργασία της αναφοράς και την επιλογή μορφοτύπου για τις εξαγωγές της (json, XLSX, PDF ή DOCX),
  • η γραμμή εργαλείων φιλτραρίσματος πληροφοριών επιτρέπει την επιλογή των πληροφοριών που πρέπει να αναφέρονται, ανάλογα με τον προσδιορισμό ή την αξιολόγησή τους,
  • το πλαίσιο προεπισκόπησης παρέχει επισκόπηση των πληροφοριών που περιέχονται στις τελικές εκθέσεις. Παρέχει επίσης έναν απλό επεξεργαστή για να προσθέσετε ή να αφαιρέσετε πληροφορίες στην αναφορά. Σημειώστε ότι η τελική απόδοση μπορεί να διαφέρει ανάλογα με την επιλεγμένη μορφή εξαγωγής.

Επεξεργασία/εξαγωγή γραμμής εργαλείων

Γραμμή εργαλείων επεξεργασίας/εξαγωγής για την επεξεργασία πληροφοριών των εκθέσεων και για την εξαγωγή σε δεδομένη μορφή:

Ανοίξτε έναν απλό επεξεργαστή στο πλαίσιο προεπισκόπησης για την προσθήκη ή την κατάργηση πληροφοριών στην παραγόμενη αναφορά. Σημειώστε ότι οι τροποποιημένες πληροφορίες θα είναι ορατές μόνο κατά την εξαγωγή σε pdf ή docx. Η επιλογή νέων φίλτρων από τη γραμμή εργαλείων φιλτραρίσματος πληροφοριών διαγράφει όλες τις τρέχουσες τροποποιήσεις.
Εξαγωγή ανεπεξέργαστων πληροφοριών μιας δεδομένης ανάλυσης, σεναρίου ή κάρτας χρησιμοποιώντας τη μορφή JavaScript Object Notation (JSON).
Εξαγωγή μη επεξεργασμένων πληροφοριών μιας δεδομένης ανάλυσης, σεναρίου ή κάρτας σε υπολογιστικά φύλλα με τη μορφή Office Open XML (XLXS).
Εξαγωγή μορφοποιημένων πληροφοριών μιας δεδομένης ανάλυσης, σεναρίου ή κάρτας σε μορφή φορητού εγγράφου (PDF).
Εξαγωγή μορφοποιημένων πληροφοριών μιας δεδομένης ανάλυσης, σεναρίου ή κάρτας σε επεξεργάσιμο Office Open XML (DOCX).

Οι μορφοποιημένες πληροφορίες αντιστοιχούν στη δημιουργία αναπαράστασης HTML μιας αναφοράς με βάση ένα πρότυπο σε μορφή PUG. Νέα πρότυπα μπορούν να εισαχθούν στην ενότητα Πρότυπο της αριστερής γραμμής πλοήγησης και να επιλεγούν στη γραμμή εργαλείων φιλτραρίσματος πληροφοριών.

Γραμμή εργαλείων φιλτραρίσματος πληροφοριών

Η γραμμή εργαλείων φιλτραρίσματος πληροφοριών επιτρέπει την επιλογή ανάλυσης, σεναρίων και καρτών ανάλογα με την ταυτοποίησή τους ή την αντίστοιχη αξιολόγησή τους σύμφωνα με τις επιλογές ή ένα υπόδειγμα:

Επιλογή προτύπου Χρήση προτύπου έχει εισαχθεί από την ενότητα Πρότυπο στην αριστερή πλευρά της γραμμής πλοήγησης.
Επιλογή ετικετών για αναφορά Κατά την αναφορά μιας πλήρους ανάλυσης, επιτρέπει την επιλογή ενός συνόλου σεναρίων ή όλων των σεναρίων που θα συμπεριληφθούν στην αναφορά.
Επιλογή καρτών για αναφορά Κατά την αναφορά μιας ανάλυσης ή ενός σεναρίου, επιτρέπει την επιλογή ενός συνόλου καρτών ή όλων των καρτών που θα συμπεριληφθούν στην αναφορά.
Επιλογή αξιολογήσεων για αναφορά Κατά την αναφορά μιας ανάλυσης, ενός σεναρίου ή μιας κάρτας, επιτρέπει την επιλογή αξιολογήσεων ή σχολίων που θα συμπεριληφθούν στην έκθεση. Σημειώστε ότι αυτή η επιλογή ισχύει επίσης για γραμμές σε cookies, τοπική αποθήκευση, beacons και testssl.

Οι επιλογές προτύπου και φιλτραρίσματος είναι αμοιβαία συμπεριληπτικές. Ως εκ τούτου, τα πρότυπα μπορούν να είναι πλήρεις πληροφορίες σχετικά με την ανάλυση, τα σενάρια και τις κάρτες που θα μπορούσαν αργότερα να επιλεγούν μέσω των επιλογών φιλτραρίσματος.

Επεξεργασία νέων προτύπων

Ωστόσο, η μόνη υποστηριζόμενη σύνταξη για το πρότυπο είναι το PUG. Οι αναφορές που δημιουργούνται από ένα πρότυπο αντιστοιχούν στο αντικείμενο αντικειμένου δεδομένων που περιέχεται στο json μιας ανάλυσης, ενός σεναρίου ή μιας κάρτας στους αντίστοιχους ντόπιους στη σύνταξη pug.

Μπορείτε να ανατρέξετε στην ενότητα Γλωσσική Αναφορά της επίσημης ιστοσελίδας της για να λάβετε πλήρεις πληροφορίες στη σύνταξή της.

ΗπεριγραφήΤο πρότυπο παρέχει ένα σύνολο ενεργειών για την εισαγωγή, επεξεργασία ή αντιγραφή υπαρχόντων προτύπων:

Εισαγάγετε ένα πρότυπο στη σύνταξη με ένα όνομα και έναν συντάκτη για τη μεταγενέστερη χρήση του σε αναφορές.
Αντίστοιχα κατεβάστε και διαγράψτε τα εισαγόμενα πρότυπα

Ως πρώτη εισαγωγή για τη δημιουργία νέων προτύπων, συνιστούμε τη λήψη μιας ανάλυσης, ενός σεναρίου ή μιας ετικέτας από τη γραμμή εργαλείων Επεξεργασία/εξαγωγή σε ακατέργαστη μορφή JSON για την εύρεση αντίστοιχων καταχωρήσεων.

Πολλά online Pug Testing Tool επιτρέπουν την αντιγραφή / επικόλληση του προτύπου PUG και των πληροφοριών JSON για την απόδοση της τελικής HTML. Το προεπιλεγμένο πρότυπο μπορεί επίσης να εξαχθεί ως αναφορά.
